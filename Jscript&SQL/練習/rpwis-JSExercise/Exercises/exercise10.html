請產出一個左右找碴的小遊戲，請選出左側多餘的那張logo(使用Javascript或JQuery皆可以)

  • 初始顯示兩張圖片於左側，右側一張
  • 請於左側選擇多出來的那張圖片
  • 選擇正確，兩側圖片各增加一張，且重新產出圖片新位置，繼續遊戲
  • 選擇錯誤，跳出Game over!訊息，並停止遊戲
    • Hint:
    JQuery - add()、children()、clone()、append()、last()、remove()、click()、empty()、unbind()
    Javascript - getElementById、createElement、appendChild、cloneNode、lastChild、removeChild、onclick、hasChildNodes
